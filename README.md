
# Simple C++ Project

A structured C++ project developed as part of Semester 4 coursework.  
This repository demonstrates fundamental and intermediate C++ programming concepts including object-oriented programming, modular design, and problem solving using structured logic.

---

## 📌 Project Overview

This project is built to strengthen core C++ concepts such as:

- Object-Oriented Programming (OOP)
- Functions and Modular Programming
- File Handling (if applicable)
- Data Structures (if used)
- Input/Output Handling
- Control Structures (loops, conditionals)
- Class Design and Encapsulation

The project follows a clean structure for maintainability and scalability.

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Adarshthakur-850/simple-project-.git
cd simple-project-
````

---

### 2️⃣ Compile the Program

If using GCC:

```bash
g++ src/main.cpp -o program
```

If multiple source files:

```bash
g++ src/*.cpp -o program
```

---

### 3️⃣ Run the Executable

On Windows:

```bash
program.exe
```

On Linux/macOS:

```bash
./program
```
